# MultiPDF reader using Python, GPT-4, and LangChain
### Dependencies
* pipenv/pyenv
* python 3.12
* GPT api key
* Huggingface api key

# How to run
install pipenv and run `pipenv install`
then run the applicaiton using `python3 main.py for cli version`
to run the frontend use `streamlit run main.py`
